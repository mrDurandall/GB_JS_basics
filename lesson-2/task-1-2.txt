Задание 1

Дан код:
var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1
c = (2+ ++a); alert(c);      // 5
d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                    // 3

Почему код даёт именно такие результаты?


Оператор инкремента ++ увеличивает переменную, к которой применен на 1.
Префиксная и постфиксная записи отличаются возвращаемым значением.
Префиксная возвращает значение переменной после увеличения, 
а постфиксная - до увеличения.
Таким образом, в приведенном примере:
c = ++a; alert(c); 	// переменная a увеличивается на 1 и принимает значение 2
					// в переменную c возвращется значение a после увеличения, т.е. тоже 2
					
d = b++; alert(d); 	// переменная b увеличивается на 1 и принимает значение 2
					// в переменную d возвращется значение b до увеличения, т.е. 1
c = (2+ ++a); alert(c);	// переменная a увеличивается еще на 1 и принимается значение 3
						// ++a возвращает значение после увеличения - 3.
						// c принимает значение суммы 3 и 2
d = (2+ b++); alert(d);	// переменная b увеличивается еще на 1 и принимается значение 3
						// b++ возвращает значение до увеличения - 2.
						// d принимает значение суммы 2 и 2
alert(a);	// выводят текущие значения a и b
alert(b);


Задание 2

Чему будет равен x в примере ниже?
var a = 2;
var x = 1 + (a *= 2);

(a *= 2) работает как a = a * 2 т.е. a = 2 * 2 = 4
таким образом var x = 1 + (a *= 2); сводится к x = 1 + 4
x примет значение 5, a при этом будет равно 4
